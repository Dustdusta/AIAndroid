# AIAndroid is:
一个开源Android聊天框架，使用RecyclerView实现。<br/>
已知在Android 5.0以下的系统的Dialogue会出现透明度的bug，在Android 2.0以下按钮好像没一个按得动。<br/>
我猜是CardView的锅。。
